# Personal Portfolio Website

A clean, responsive, and professional personal portfolio website built to showcase my academic projects, skills, and resume. 

🔗 **Live Demo:** [Insert your Vercel URL here]

## 🚀 Features
- **Responsive Design:** Optimized for mobile, tablet, and desktop viewing.
- **Dynamic Projects Section:** Displays academic projects with direct links to GitHub repositories and image previews.
- **Direct Resume Viewing:** Integrated Google Drive link to view my up-to-date resume without forced downloads.
- **Interactive Modals:** Custom lightboxes for clean project image displays.

## 🛠️ Built With
- **HTML5** - For structured, semantic markup.
- **CSS3** - Custom properties (variables), Flexbox, Grid, and keyframe animations for polished visuals.
- **Vanilla JavaScript** - For dynamic project rendering and modal interactivity.

## 📁 Repository Structure
- `index.html` - The main structure and content of the portfolio.
- Images and screenshots used for project mockups.
